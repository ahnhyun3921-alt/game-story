# 이미지 파일 안내

이 폴더에는 다음 이미지 파일들이 필요합니다:

## 필수 이미지 목록

### 아이콘 (24x24px ~ 48x48px)
- `logo.png` - 로고 이미지
- `icon-notification.png` - 알림 벨 아이콘
- `icon-direct.png` - 직접 작성 아이콘
- `icon-ai.png` - AI 작성 아이콘
- `icon-upload.png` - 업로드 아이콘

### 일러스트 (200x200px ~ 400x400px)
- `illust-upload.png` - 업로드 캐릭터 일러스트
- `illust-direct.png` - 직접 작성 캐릭터 일러스트 (노트북 든 캐릭터)
- `illust-ai.png` - AI 작성 캐릭터 일러스트 (안경 쓴 물방울 캐릭터)

### 배너 이미지
- `select-banner.png` - 시나리오 작성 선택 페이지 배너 (유령 캐릭터)
- `start-banner.png` - 홈 페이지 배너 (4개 캐릭터: 물방울, 유령, 새싹, 불꽃)
- `team-banner.png` - 팀원 찾기 배너 (망원경 든 불꽃 캐릭터)

### 역할 선택 이미지 (300x300px ~ 400x400px)
- `role-writer.png` - 작가 역할 이미지 (펜 든 불꽃 캐릭터)
- `role-creater.png` - 제작자 역할 이미지 (안경 쓴 유령 캐릭터)

## 임시 방법

이미지가 없는 경우, 다음 방법으로 임시 처리 가능합니다:

### 방법 1: 무료 아이콘 사이트 사용
- [Flaticon](https://www.flaticon.com/)
- [Icons8](https://icons8.com/)
- [Ionicons](https://ionic.io/ionicons)

### 방법 2: 이모지 사용
CSS에서 이미지 대신 이모지를 사용할 수 있습니다:
- 🔔 (알림)
- 📝 (직접 작성)
- 🤖 (AI)
- 📤 (업로드)

### 방법 3: SVG 생성
간단한 SVG 아이콘을 직접 생성하여 사용할 수 있습니다.

---

## 디자인 가이드

- **컬러**: #E85F5C (Primary), #333 (Text), #f5f5f5 (Background)
- **스타일**: 둥근 모서리, 부드러운 그림자, 친근한 캐릭터
- **파일 포맷**: PNG (투명 배경), SVG (벡터)
- **최적화**: 웹 최적화된 이미지 사용 (압축)
