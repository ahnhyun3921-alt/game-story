## 🎁 제공된 기능 상세

### 홈 페이지
```
- 4개 캐릭터 배너
- 장르 필터 드롭다운
- 검색 입력창
- 시나리오 카드 8개 (그리드 레이아웃)
- 각 카드: 제목, 작가, 설명, 태그, 평점
- 호버 애니메이션
```

### 팀원 찾기
```
- 망원경 든 캐릭터 배너
- 작가 유형 필터
- 검색 입력창
- 팀원 카드 8개
- 각 카드: 이름, 역할, 설명, 스킬 뱃지, 아바타
```

### 시나리오 작성
```
- 3가지 방식 선택 카드
  1. 직접 작성 (텍스트 에디터)
  2. AI 작성 (프롬프트 입력)
  3. 업로드 (파일 선택)
- 각 방식별 전용 페이지
- 캐릭터 일러스트
- 제출 버튼
```

### 회원가입
```
- Step 1: 정보 입력
  - 아이디 (중복 확인 버튼)
  - 비밀번호 (유효성 힌트)
  - 닉네임 (중복 확인 버튼)
  - 비밀번호 재입력 (일치 확인)
- Step 2: 역할 선택
  - 제작자 카드 (어두운 배경)
  - 작가 카드 (어두운 배경)
  - 선택 시 테두리 강조
```

### 로그인
```
- 간단한 폼
- 아이디 입력
- 비밀번호 입력
- 로그인 버튼
- 회원가입 링크
```

### 알림
```
- 헤더 벨 아이콘
- 클릭 시 드롭다운
- 알림 4개 표시
- 날짜 + 메시지
- 호버 효과
```
